HorseFly

What is HorseFly?

HorseFly is a a delivery system where a moving truck (with packages) and a drone are autonomously coordinating to deliver the packages to the customer homes in approximately minimum time.

The algorithm uses OPTICS density clustering and kdTree based nearest neighbour search.


How to visualize?

● Click on Draw Path to draw a delivery path.
● Click on Draw Homes to draw homes for package delivery.
● Click on Run! to run the algorithm and see the animation.
