HorseFly

What is HorseFly?

HorseFly is a a delivery system where a moving truck (with packages) and a drone are autonomously coordinating to deliver the packages in the customer homes in approximately minimum time.

How to visualize?

Click on Draw Path to draw a delivery path.
Click on Draw Homes to draw a bunch of homes for package delivery.
Click on Run to run the algorithm to see the visualization.
